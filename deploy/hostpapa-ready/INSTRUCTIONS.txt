📤 ÉTAPES D'INSTALLATION :

1. CONNECTEZ-VOUS À CPANEL HOSTPAPA
   - Allez sur https://cpanel.hostpapa.com
   - Connectez-vous avec vos identifiants

2. OUVREZ FILE MANAGER
   - Cliquez sur "File Manager" dans cPanel
   - Naviguez vers le dossier "public_html"

3. NETTOYEZ LE DOSSIER
   - Supprimez tous les fichiers existants dans public_html
   - (Sauvegardez d'abord si vous avez des fichiers importants)

4. UPLOADEZ LES FICHIERS
   - Sélectionnez TOUS les fichiers de ce dossier
   - Glissez-déposez dans public_html
   - OU utilisez le bouton "Upload" de cPanel

5. VÉRIFIEZ LES PERMISSIONS
   - Fichiers : 644
   - Dossiers : 755
   - (normalement configuré automatiquement)
