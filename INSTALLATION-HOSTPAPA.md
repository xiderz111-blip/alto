# 🚀 Installation Alt0 sur HostPapa - Guide Complet\n\nVotre clone moderne de Bolt.new est prêt à être déployé sur HostPapa !\n\n## ⚡ Installation Automatique en Une Commande\n\n```bash\ncd /app/deploy\n./deploy-to-hostpapa.sh\n```\n\n**C'est tout !** Le script vous guidera à travers :\n1. Configuration de votre domaine HostPapa\n2. Build automatique de l'application\n3. Upload vers votre hébergement\n\n## 📋 Ce dont vous avez besoin\n\n### Informations HostPapa\n- 🌐 **Votre domaine** (ex: monsite.com)\n- 👤 **Nom d'utilisateur FTP** (trouvé dans cPanel)\n- 🔒 **Mot de passe FTP** (configuré dans cPanel)\n\n### Types d'hébergement supportés\n- ✅ **Hébergement partagé** - Version statique avec toutes les fonctionnalités\n- ✅ **VPS/Serveur dédié** - Version complète avec backend\n\n## 🎯 Fonctionnalités déployées\n\n### ✨ Interface utilisateur complète\n- Page d'accueil avec gradients animés\n- Génération de projets par IA (mockée)\n- Éditeur de code avec coloration syntaxique\n- Prévisualisation responsive (Desktop/Tablet/Mobile)\n- Navigation fluide\n\n### 🛠️ Fonctionnalités techniques\n- Application React optimisée\n- Routing côté client\n- Cache des fichiers statiques\n- Compression gzip automatique\n- Configuration Apache (.htaccess)\n\n## 📂 Structure des fichiers déployés\n\n```\npublic_html/\n├── index.html          # Application principale\n├── static/\n│   ├── css/           # Styles optimisés\n│   └── js/            # JavaScript minifié\n├── .htaccess          # Configuration Apache\n└── config.js          # Configuration de production\n```\n\n## 🔧 Options avancées\n\n### Redéploiement rapide\n```bash\n./deploy-to-hostpapa.sh --skip-config\n```\n\n### Build seulement (sans upload)\n```bash\n./build-for-hostpapa.sh shared  # pour hébergement partagé\n./build-for-hostpapa.sh vps     # pour VPS\n```\n\n### Upload manuel\n```bash\n# Configurer les variables\nexport HOSTPAPA_HOST=\"votredomaine.com\"\nexport HOSTPAPA_USER=\"votre_username\"\nexport HOSTPAPA_PASS=\"votre_password\"\n\n# Upload\n./hostpapa-upload.sh shared\n```\n\n## 🎨 Personnalisation\n\n### Changer l'URL de production\nModifiez `/app/frontend/.env.production` :\n```\nREACT_APP_BACKEND_URL=https://votredomaine.com\n```\n\n### Personnaliser les couleurs/branding\nLes fichiers principaux à modifier :\n- `/app/frontend/src/pages/HomePage.jsx` - Page d'accueil\n- `/app/frontend/src/index.css` - Styles globaux\n- Logo \"A0\" peut être changé dans le header\n\n## 🚨 Dépannage\n\n### Le site ne charge pas\n1. ✅ Vérifiez que les fichiers sont dans `public_html`\n2. ✅ Vérifiez le fichier `.htaccess`\n3. ✅ Consultez les logs d'erreur dans cPanel\n\n### Erreur lors de l'upload\n1. ✅ Vérifiez vos identifiants FTP\n2. ✅ Testez la connexion FTP manuellement\n3. ✅ Vérifiez l'espace disque disponible\n\n### Problème de build\n```bash\n# Nettoyer et rebuilder\ncd /app/frontend\nrm -rf build node_modules\nnpm install\nnpm run build\n```\n\n## 📞 Support\n\n### Logs de débogage\n```bash\n# Voir les logs de build\ncat /app/deploy/*.log\n\n# Tester la connexion FTP\nlftp -u username hostname\n```\n\n### Ressources HostPapa\n- [Documentation cPanel HostPapa](https://hostpapa.com/knowledgebase/)\n- [Support technique HostPapa](https://hostpapa.com/contact/)\n\n## 🔄 Mises à jour\n\nPour mettre à jour votre site après modifications :\n```bash\ncd /app/deploy\n./deploy-to-hostpapa.sh --skip-config\n```\n\n## 🎉 Après le déploiement\n\n1. **Testez votre site** sur https://votredomaine.com\n2. **Vérifiez toutes les fonctionnalités** :\n   -
